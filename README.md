# LMSlecDownloader

명지대학교 LMS 사용자 전용 강의 영상 다운로더

## 🏁 Getting Started

### 0. Prerequisites

- Node.js

### 1. Git clone

```shell
git clone https://github.com/13circle/LMSlecDownloader.git
```

### 2. Start React server

```shell
yarn install   # or npm install
yarn start     # or npm start
```
